## Azure Focus

Every practical uses Azure services:
- **Azure Load Balancer / Front Door** — load balancing & CDN
- **Azure Cache for Redis** — caching layer
- **Azure Service Bus / Event Grid** — messaging & events
- **Azure API Management** — API gateway & rate limiting
- **Azure Cosmos DB** — globally distributed NoSQL
- **Azure Blob Storage** — object/blob storage
- **Azure Cognitive Search** — full-text search
- **Azure Monitor / Application Insights** — observability
